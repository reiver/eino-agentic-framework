Retrieval Augmented Generation (RAG)

retriever.Retriever

by Charles Iliya Krempeaux

Retrieval-Augmented Generation (RAG) is an AI technique that retrieves relevant information from external knowledge bases and uses that to improve the context of the prompt gives to the LLM. The hope is that this helps the LLM generate a better response.

With the Eino Agentic Framework, this is done with the Eino retriever.Retriever interface.
