Prompt Template

a reusable, structured prompt blueprint (for LLMs) with placeholders that can be dynamically filled with different inputs

by Charles Iliya Krempeaux

The Eino Agentic Framework has built-in support for prompt templates.

Prompt Templates are also sometimes called: custom prompt, instruction templates, prompt blueprints, prompt patterns, prompt recipe, prompt scaffolds, prompt schemas, prompt skeletons, prompt specs, prompt specifications, etc.

(Claude Code seems to call prompt templates: commands, and slash commands.)

In Eino, prompt templates are represented with the prompt.ChatTemplate interface:
